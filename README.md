# Forensic System 🔍🛡️

A powerful and intuitive mobile application built with **Flutter**, designed to detect digital manipulation and forgery in multimedia files (Images and Audio). 

This project integrates modern **Digital Forensics** techniques with **Artificial Intelligence (AI)** to analyze files, verify media authenticity, and uncover hidden manipulations, providing a reliable tool for security and forensic analysis.

## ✨ Key Features
* **Image Forgery Detection:** Analyzes images to detect tampering, pixel manipulation, and metadata inconsistencies.
* **Audio Analysis:** Scans audio files to identify splicing, deepfakes, or unusual frequency manipulations using AI models.
* **Real-time Results:** Fast and efficient processing providing detailed forensic reports directly on your mobile device.
* **User-Friendly Interface:** A clean, responsive, and easy-to-navigate UI built specifically to streamline the forensic investigation process.
* **Cross-Platform:** Works seamlessly on both Android and iOS devices.
